This patch resolves the offline installation of version 1.18 of ArcGIS Experience Builder.
Step: 
1. Download the [1.18 Developer edition](https://developers.arcgis.com/experience-builder/guide/downloads/)
2. Download the new 1.18 node cache package from [this SDK repository](https://github.com/Esri/arcgis-experience-builder-sdk-resources/archive/refs/heads/master.zip) 
3. Browse to the patch directory under the unzipped SDK resource (./patches/1.18/exb-1.18-node_modules-cache.zip)
4. Or instead of step 2-3, you can download the new 1.18 node cache from [developer site](https://developers.arcgis.com/experience-builder/guide/downloads/).
2. Follow the offline installation guide https://developers.arcgis.com/experience-builder/guide/install-guide/#install-offline to setup.
